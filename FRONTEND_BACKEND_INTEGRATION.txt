--------------------------------------------------
2.1 AUTHENTICATION & USER ENDPOINTS
--------------------------------------------------

POST   /api/auth/register/
Body: { username, email, password, institution, role }
Response: { user_id, token, user_data }
Açıklama: Yeni kullanıcı kaydı

POST   /api/auth/login/
Body: { username, password }
Response: { token, user_data }
Açıklama: Kullanıcı girişi

POST   /api/auth/logout/
Headers: { Authorization: "Token xxx" }
Response: { success: true }
Açıklama: Kullanıcı çıkışı

GET    /api/user/profile/
Headers: { Authorization: "Token xxx" }
Response: { id, name, email, institution, role, bio, interests }
Açıklama: Kullanıcı profil bilgilerini getir

PUT    /api/user/profile/
Headers: { Authorization: "Token xxx" }
Body: { name, bio, institution, role }
Response: { success: true, updated_user }
Açıklama: Profil bilgilerini güncelle

PUT    /api/user/interests/
Headers: { Authorization: "Token xxx" }
Body: { interests: ["AI", "ML", ...] }
Response: { success: true, interests }
Açıklama: İlgi alanlarını güncelle

--------------------------------------------------
3.1 DASHBOARD (views/Dashboard.tsx)
--------------------------------------------------

Şu an:
- Mock data kullanıyor (MOCK_CONTENT, NEWS_ITEMS)
- State: savedIds (local state)

Backend Entegrasyonu:
1. Component mount'ta:
   - GET /api/content/latest/ → En son içerikleri çek
   - GET /api/news/ → Haberleri çek
   - GET /api/saved/ → Kaydedilenleri çek (savedIds state'ini doldur)

2. toggleSave fonksiyonu:
   - Eğer saved ise: DELETE /api/saved/{id}/
   - Değilse: POST /api/saved/ with { content_id }

3. Arama inputu:
   - Değişiklikte: /search sayfasına yönlendir

Örnek Implementation:
```typescript
useEffect(() => {
    // Fetch latest content
    fetch('http://localhost:8000/api/content/latest/?limit=10', {
        headers: { 'Authorization': `Token ${localStorage.getItem('token')}` }
    })
    .then(res => res.json())
    .then(data => setContent(data));
    
    // Fetch saved IDs
    fetch('http://localhost:8000/api/saved/', {
        headers: { 'Authorization': `Token ${localStorage.getItem('token')}` }
    })
    .then(res => res.json())
    .then(data => {
        const ids = new Set(data.map(item => item.content_id));
        setSavedIds(ids);
    });
}, []);
```

3.2 SEARCH DASHBOARD (views/SearchDashboard.tsx)
--------------------------------------------------

Şu an:
- Mock data kullanıyor
- Filtreler sadece UI'da var (çalışmıyor)

Backend Entegrasyonu:
1. Component mount + filter değişikliklerinde:
   - GET /api/search/?q={query}&type={type}&source={source}&date={date}

2. Filter state management:
   - contentTypeFilter, sourceFilter, dateFilter state'leri
   - Her değişiklikte API call

3. Pagination:
   - GET /api/search/?page={pageNum}

Örnek Implementation:
```typescript
const [filters, setFilters] = useState({
    query: '',
    type: [],
    source: [],
    dateRange: 'all'
});

useEffect(() => {
    const params = new URLSearchParams();
    if (filters.query) params.append('q', filters.query);
    filters.type.forEach(t => params.append('type', t));
    
    fetch(`http://localhost:8000/api/search/?${params}`, {
        headers: { 'Authorization': `Token ${localStorage.getItem('token')}` }
    })
    .then(res => res.json())
    .then(data => setResults(data.results));
}, [filters]);
```

3.6 CHAT OVERLAY (components/ChatOverlay.tsx)
--------------------------------------------------

Şu an:
- Simulated responses (setTimeout)

Backend Entegrasyonu:
1. handleSend fonksiyonu:
   - POST /api/chat/ with { message }
   - Response'u mesaj listesine ekle

Örnek Implementation:
```typescript
const handleSend = async () => {
    const userMessage = input;
    setMessages(prev => [...prev, { role: 'user', text: userMessage }]);
    
    const response = await fetch('http://localhost:8000/api/chat/', {
        method: 'POST',
        headers: {
            'Authorization': `Token ${localStorage.getItem('token')}`,
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({ message: userMessage })
    });
    
    const data = await response.json();
    setMessages(prev => [...prev, { role: 'model', text: data.reply }]);
};
```

4. VERİ AKIŞI VE STATE YÖNETİMİ
================================================================================

4.1 AUTHENTICATION FLOW
------------------------
1. User → Login Page
2. POST /api/auth/login/ → Get token
3. Store token in localStorage
4. All API requests → Include token in header
5. If 401 error → Redirect to login

4.2 GLOBAL STATE (Önerilen)
---------------------------
Context API veya Zustand kullan:

AuthContext:
- user: User | null
- token: string | null
- login(username, password)
- logout()
- isAuthenticated: boolean

ContentContext:
- savedContent: Set<number>
- toggleSave(id)
- refreshSaved()

4.3 API REQUEST INTERCEPTOR (Önerilen)
--------------------------------------
Tek bir API utility oluştur:

// src/utils/api.ts
const BASE_URL = 'http://localhost:8000/api';

export const api = {
    get: (endpoint) => fetch(`${BASE_URL}${endpoint}`, {
        headers: { 'Authorization': `Token ${getToken()}` }
    }),
    post: (endpoint, data) => fetch(`${BASE_URL}${endpoint}`, {
        method: 'POST',
        headers: {
            'Authorization': `Token ${getToken()}`,
            'Content-Type': 'application/json'
        },
        body: JSON.stringify(data)
    }),
    // ... put, delete
};

5. API SERVICE IMPLEMENTATION ÖRNEĞİ
================================================================================

// src/services/contentService.ts
export const contentService = {
    getLatest: () => api.get('/content/latest/'),
    search: (params) => api.get(`/search/?${new URLSearchParams(params)}`),
    getById: (id) => api.get(`/content/${id}/`),
    save: (id) => api.post('/saved/', { content_id: id }),
    unsave: (id) => api.delete(`/saved/${id}/`),
};

// src/services/userService.ts
export const userService = {
    getProfile: () => api.get('/user/profile/'),
    updateProfile: (data) => api.put('/user/profile/', data),
    updateInterests: (interests) => api.put('/user/interests/', { interests }),
};

// src/services/authService.ts
export const authService = {
    login: (username, password) => 
        api.post('/auth/login/', { username, password }),
    register: (data) => api.post('/auth/register/', data),
    logout: () => api.post('/auth/logout/', {}),
};

7. ÖNEMLİ NOTLAR
================================================================================

1. CORS: Backend'de CORS_ALLOWED_ORIGINS içinde frontend URL'i olmalı
2. Authentication: Token-based (Django REST framework token)
3. Error Handling: Tüm API calls'larda try-catch ve error states
4. Loading States: Her API call için loading indicator
5. Pagination: Large datasets için pagination implement et
6. Caching: Consider React Query veya SWR for caching
7. Environment Variables: API URL'leri .env dosyasında tut

CORS Settings (Backend):
CORS_ALLOWED_ORIGINS = [
    "http://localhost:5173",  # Vite
    "http://localhost:3000",  # Alternative
]

Environment Variables (Frontend):
VITE_API_BASE_URL=http://localhost:8000/api
